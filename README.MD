# Ecuación de la Subjetividad

Autor: **Enrique Bernal**

---

## **Propósito**
Esta ecuación busca cuantificar la experiencia subjetiva y emocional de una persona, integrando factores fisiológicos, emocionales, cognitivos y contextuales en un solo marco matemático adaptable.

---

## **Ecuación General**

**S<sub>Enrique</sub> = α × (E + M + C + F + Ψ + Δf + β + ε + η + ...)**

Donde:
- **S<sub>Enrique</sub>** = Subjetividad total (Enrique Bernal)
- **α** = Factor de ajuste personal (puede ser la proporción áurea φ, una constante elegida o autocalibrada)
- **E** = Estado emocional (escala cuantificada: ánimo, ansiedad, irritabilidad, energía)
- **M** = Estado mental (claridad, atención, creatividad, percepción)
- **C** = Contexto (factores externos, ambiente, personas)
- **F** = Estado físico (dolor, fatiga, sueño, síntomas)
- **Ψ** = Frecuencia/función de onda cerebral (ex: ondas beta)
- **Δf** = Variación de frecuencia (diferencia entre estados y momentos)
- **β** = Beta: nivel de excitación/alerta
- **ε, η, ...** = Otros factores relevantes (espirituales, filosóficos, motivacionales)

---

---

## **Interpretación**
La ecuación sirve como:
- Herramienta de autoconocimiento y seguimiento personal.
- Base para modelos predictivos de bienestar y estados mentales/emocionales.
- Puente entre datos objetivos (frecuencias, síntomas) y experiencia subjetiva (emociones, percepción).
- Inspiración para estudios transdisciplinarios en psicología, neurociencia y filosofía.

---

## **Nota**
La ecuación es adaptable: puedes agregar/quitar factores según tu experiencia y necesidades, y ajustar α para personalización.
